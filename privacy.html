개인정보 처리방침

최종 업데이트: 2026년 6월

1. 수집하는 정보

본 봇(이하 "강화 RPG")은 게임 진행을 위해 다음 정보를 저장합니다.

  • 디스코드 사용자 ID — 게임 데이터를 식별·연결하기 위한 숫자 ID
  • 게임 진행 데이터 — 보유 골드/아이템, 강화·레벨·환생 등 성장 상태, 칭호·업적, 길드/랭킹 정보 등
  • 커스터마이즈 — 닉네임, 조력자 이름, 업로드한 조력자 이미지(선택)

메시지 내용은 명령어 처리에만 사용되며 별도로 영구 저장하지 않습니다.

2. 정보의 이용 목적

  • 게임 진행 상태의 저장 및 불러오기
  • 랭킹·길드·장터 등 유저 간 상호작용 기능 제공
  • 서비스 운영·오류 진단 및 어뷰징 방지

3. 제3자 제공 및 외부 서비스

수집된 정보는 외부에 판매·제공되지 않습니다. 단, 다음 서비스를 이용합니다.

  • Discord API — 봇 동작에 필수
  • Anthropic Claude API — 일부 AI 기능(신탁·AI 제작) 사용 시, 입력한 질문/설명 텍스트가 처리됩니다. 개인 식별 정보는 전송하지 않습니다.

4. 보관 및 삭제

게임 데이터는 서비스 제공 기간 동안 보관됩니다. 본인의 데이터 삭제를 원하시면 아래 연락처(지원 서버)로 요청해 주세요. 요청 시 해당 사용자 ID의 게임 데이터를 삭제합니다.

5. 문의

개인정보 관련 문의는 지원(서포트) 디스코드 서버를 통해 접수할 수 있습니다.

※ 본 방침은 서비스 정책 변경에 따라 갱신될 수 있습니다.
